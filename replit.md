# Удалённый рабочий стол

Простая программа удалённого доступа к компьютеру — аналог AnyDesk.

## Как это работает

1. **Друг** скачивает папку с проектом и запускает `ЗАПУСТИТЬ.bat` (Windows) или `ЗАПУСТИТЬ.sh` (Mac/Linux)
2. Скрипт автоматически устанавливает Python и все зависимости
3. Запускается сервер и создаётся публичная ссылка через ngrok
4. Друг отправляет ссылку тебе
5. Ты открываешь ссылку в браузере и видишь его экран, можешь управлять мышью и клавиатурой

## Файлы проекта

- `ЗАПУСТИТЬ.bat` — файл запуска для Windows (друг запускает его)
- `ЗАПУСТИТЬ.sh` — файл запуска для Linux/Mac
- `launcher.py` — установщик зависимостей и запускатель с ngrok-туннелем
- `app.py` — основной Flask-сервер с захватом экрана и управлением вводом
- `templates/` — HTML-страницы (главная, хост, просмотрщик)

## Технологии

- Flask + Flask-SocketIO — веб-сервер с WebSocket
- mss + Pillow — захват экрана
- pyautogui — управление мышью и клавиатурой
- pyngrok — публичный туннель для доступа из интернета
